# Demo Builder

Demo Builder creates branded, configurable agent demo HTML files from uploaded or pasted JSON.

## Inputs

- Content JSON for scenarios, messages, documents, and logs.
- Sizing/prerequisites JSON.
- Per-scenario glossary JSON.
- Branding and visual configuration from the UI.

## Output

Generated demos are saved in `Outputs/` as HTML files.

The generated demo supports:

- scenario selection
- progressive chat playback
- document reveal
- grouped agent logs
- glossary modal
- simulation speed controls
- pause/resume
- chat-style interface with avatars
